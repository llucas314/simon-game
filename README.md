## How to Run

1. Fork and clone this repository.
2. Open index.html in your browser.
3. The lights can be either clicked on or controlled by the keys.

|Key|Light|
|:---:|:---:|
|<kbd>w</kbd>| `Red Light`|
|<kbd>e</kbd>| `Blue Light`|
|<kbd>d</kbd>| `Green Light`|
|<kbd>s</kbd>| `Yellow Light`|
